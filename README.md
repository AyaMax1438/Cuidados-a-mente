# Cuidados-a-mente
Aqui vai ficar todos os arquivos do projeto

## O que é?
"Cuidados-a-mente" é um aplicativo que visa digitalizar os cuidados de um idoso com Alzheimer. Além de ter um banco de memórias, o aplicativo conta com rotinas, medicação e agendamento de consultas, todas registradas pelo responsável.

## Por que?
Nosso objeto é facilitar esse cuidado, monitorando as rotinas e atividades do idoso. Assim, o familiar pode se atentar ao que o idoso faz de forma contínua e moderna. Além do mais, o idoso consegue fortificar laços familiares com a função "banco de memórias".
Vale lembrar que alguns dos nossos integrantes passaram pela situação de cuidar de um idoso com Alzheimer.

## Quem?
Temos dois público-alvo:
- Os idosos: entre 2 milhões de brasileiros que vivem com demência, 70% dos casos é Alzheimer;
- os cuidadores/familiares: dos cuidadores, 80% das vezes são mulheres de família sobrecarregada.
Nosso aplicativo visa diminuir a preocupação em relação as atividades, como médico, por não ter que olhar o celular toda hora, já que teremos alarmes inteligentes.

## Como?
São 3 perfis:
- Idoso: é um perfil base com informações necessárias, como lista de contato de familiares e lista de remédio. O idoso terá acesso tanto a rotina quanto o "banco de memórias", mas não vai poder editar nenhum dos dois. Também terá um "botão de alerta" que, se o idoso acionar, ligará para um dos responsáveis. O perfil terá um tóken, que será do responsável, que pode ser compartilhado com o cuidador;
- Cuidador: é um perfil "profissional" que tem a permissão dos responsáveis para ver a rotina, atendimentos médicos, lembretes e atividades do idoso. No perfil, o cuidador pode ver os horários de cada idoso que cuidará, já que pode cuidar de mais de um idoso;
- Responsável(familiar): administrador, que cria, gerencia e edita as rotinas, atividades, consultas agendados e os remédios do perfil do idoso. Quando o perfil é criado, o responsável pode criar o perfil do idoso na hora. O responsável pode compartilhar o tóken do idoso para o cuidador, caso seja outra pessoa.
